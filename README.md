# 🤖 AI Resume Analyzer

An AI-powered Resume Analyzer built using **Python**, **Flask**, and **Bootstrap**. It compares a resume with a Job Description (JD), calculates an ATS score, and identifies matched and missing skills.

---

## 🚀 Features

- 📄 Upload Resume (PDF)
- 📝 Paste Job Description
- 📊 ATS Score Calculation
- ✅ Matched Skills Detection
- ❌ Missing Skills Detection
- 🎨 Responsive Bootstrap UI
- ⚡ Fast PDF Text Extraction

---

## 🛠️ Tech Stack

- Python
- Flask
- HTML5
- CSS3
- Bootstrap 5
- PyMuPDF (fitz)
